CSE226 - Numerical Methods Sessional

This repository contains my lab work for the Numerical Methods course (CSE226). It includes implementations of various numerical algorithms commonly used for solving equations and systems of linear equations.

Methods Implemented

1. Bisection Method  
2. One-Point Iteration  
3. Newton-Raphson Method  
4. Secant Method  
5. Gauss Elimination  
6. Gauss-Jordan Elimination  
7. Gauss-Seidel Method
